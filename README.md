
## Laravel Ecommerce

A full-featured eCommerce application built with Laravel, featuring user authentication, product management, shopping cart functionality, and more.

## Feature
- User registration and authentication.
- Product listing and filtering by category.
- Shopping cart and checkout.
- Payment gateway intergration(Stripe,Paypal).
- Order management for User, Vendor and Admin.
- Admin panel for product and order management.
- Chat real-time communication for client and the seller

## Requirements
- PHP >= 8.x
- Composer
- Laravel 10.x
- MySQL or any other supported database
- Node.js & npm (for frontend assest management)
- Stripe & PayPal Api keys (for payment intergration)
- Yajra Datatable (lightweight to display data to table )
